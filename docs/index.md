# Editor Focus Mode

Editor Focus Mode hides non-relevant note fields in Anki's Browser editor so you can work with a smaller, task-focused set of fields.

## Highlights

- hides fields based on note type layout rules
- adds a toggle to temporarily reveal all fields
- supports named layouts with field ordering
- provides layout editing for the current note type
